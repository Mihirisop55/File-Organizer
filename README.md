# File-Organizer
Are all your files scrattered around in a directory??
No worries because I build a Smart File organizer.


# *Features*
Analyzes every file in the given directory
Sorts every file and creates a directory for it
If a directory already exists, it uses the existing one
Very easy to use


# Technologies Used
Python
Os
Shutil

# What I Learned
While building this project I learned:
File Handling
Moving Files
If-Else statement
Loops

# How to use
Its pretty simple contrary to its name
1. You just have to paste the file in any directory you want to sort
2. Run the file using any editor (for ex. VS Code, PyCharm, etc)
